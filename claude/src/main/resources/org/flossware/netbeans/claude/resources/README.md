# Icon Resources

## claude-icon.png

This directory should contain a `claude-icon.png` file (16x16 pixels).

You can:
1. Create a simple 16x16 icon with your favorite image editor
2. Use a placeholder icon temporarily
3. Design a custom Claude/AI themed icon

For now, you'll need to add this file manually. The icon will appear in:
- The Tools menu
- The Window menu
- The Claude chat window tab

Recommended format: PNG, 16x16 pixels
