## Setup

1. **Get API Key** (COMPLETELY FREE):
   - Visit https://platform.deepseek.com
   - Create account (NO credit card required)
   - Generate API key
   - **No quota limits!**

2. **Configure in NetBeans**:
   - Tools → Options → Advanced → DeepSeek AI
   - Enter your API key
   - Click "Test Connection"
   - Select model (default: deepseek-chat)

3. **Usage**:
   - Right-click code → "Ask DeepSeek AI"
   - Window → DeepSeek AI (open chat)
   - Perfect for unlimited coding assistance!
