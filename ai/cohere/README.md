# Cohere AI Integration for NetBeans

Integrates Cohere AI into Apache NetBeans IDE - enterprise-grade AI platform.

## Features

- 💬 Interactive AI chat window
- 🏢 Enterprise-focused AI responses
- 🔍 Code analysis and generation
- 📝 Context-aware development assistance
- ⚙️ Configurable model selection

## Free Tier ✅

**Cohere offers a FREE tier for development!**
- **Free quota**: 1,000 API calls per month
- **Free models**: `command-light`, `command-r`
- **Default model**: `command-light` (FREE, optimized)
- **Get API key**: https://dashboard.cohere.com
- No credit card required for trial/dev tier

## Setup

1. **Get API Key** (FREE for development):
   - Visit https://dashboard.cohere.com
   - Create account (no credit card needed)
   - Generate API key (Trial tier: 1000 calls/month)

2. **Configure in NetBeans**:
   - Tools → Options → Advanced → Cohere AI
   - Enter your API key
   - Click "Test Connection"
   - Select model (default: command-light - FREE)

3. **Usage**:
   - Right-click code → "Ask Cohere AI"
   - Window → Cohere AI (open chat)
   - Select code for context-aware analysis

## Available Models

### Free Tier Models ✅
- `command-light` - **FREE** (default) - Fast, efficient, great for development
- `command-r` - **FREE** - More capable, retrieval-focused

### Production Models 💰
- `command` - Standard production model
- `command-r-plus` - Most capable, enterprise features
- `command-nightly` - Latest experimental features
