## Getting Started

### Installation (5 Minutes)
```bash
mvn clean package
# Install NBM: Tools → Plugins → Downloaded
```

### Configuration (10 Minutes)
1. Choose AI provider (free options: DeepSeek, Claude, Gemini, Mistral, Cohere, OpenRouter, Perplexity)
2. Get free API key from provider
3. Tools → Options → [AI Provider]
4. Enter API key and test connection

### First Productivity Gain (<1 Hour)
- Write code, press Ctrl+Space
- Right-click code → "Ask [AI] About This Code"
- Open AI chat: Tools → Open [AI] Chat

**Total time to ROI: 20 minutes**
